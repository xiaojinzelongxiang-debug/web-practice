# web-practice

このリポジトリは、Web制作の基礎を練習しながら、ポートフォリオとして公開している学習用プロジェクトです。

「まずは1ページを作って公開する」ことを目標に、HTML / CSS / JavaScript の基本を少しずつ確認しています。

## このリポジトリでできること

- 学習中のWebページを確認できる
- コードの更新履歴（コミット）を追える
- 基礎学習の進み方を記録として残せる

- HTML5
- CSS3

## 学習目的

- HTMLとCSSで基本的なページを自力で作れるようになる
- JavaScriptを使った簡単な動きを実装できるようになる
- Git/GitHubを使って更新・管理の流れに慣れる
- 案件応募時に提示できる学習ポートフォリオを整える
